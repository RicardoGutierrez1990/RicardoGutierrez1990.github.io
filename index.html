PROJECTS

Virtual Pet Game

A fun console application that allows users to create virtual pets. The user can put them in a virtual pet shelter and interact with the pets. The user can also pick between playing with an organic pet or a robotic pet.

FizzBuzzQix

A fun and challenging exercise to put in practice Test Driven Development. Counting with a catch. For all multiples of three, the player must say "Fizz", for all multiples of five, the player will say "Buzz" and for all multiples of seven, the player will say "Qix. If a number is both a multiple of three and five the player says "FizzBuzz". If a number is both a multiple of three and seven the player says "FizzQix". If a number is both a multiple of five and seven the player says "BuzzQix". If a number is a multiple of three, five and seven the player says "FizzBuzzQix".

Guessing Game

Console application that challenges the user to guess a random number. If the player inputs the right, a congratulatory message appears, otherwise, the player has the option to keep guessing.
